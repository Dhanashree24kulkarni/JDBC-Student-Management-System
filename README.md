# Student Management System using Core Java, JDBC & Oracle Database

## Project Overview

Developed a console-based Student Management System that performs complete CRUD operations by integrating Core Java with Oracle Database using JDBC.

## Features

* Add Student Records
* View All Students
* Search Student by ID
* Update Student Information
* Delete Student Records
* Menu-Driven Console Application

## Technical Concepts Implemented

* JDBC Connectivity
* Oracle Database Integration
* PreparedStatement
* ResultSet Processing
* Exception Handling
* Object-Oriented Programming (OOP)
* DAO Design Pattern
* Database Connection Management

## Technologies Used

* Java 17
* JDBC
* Oracle Database 10g
* SQL*Plus
* Eclipse IDE

## Project Structure

com.jdbc.model  - Model Classes

com.jdbc.dao    - Database Operations

com.jdbc.util   - Database Connectivity

com.jdbc.main   - Application Entry Point

## Learning Outcomes

* Established database connections using JDBC DriverManager
* Executed SQL queries using PreparedStatement
* Retrieved data using ResultSet
* Implemented CRUD operations with Oracle Database
* Applied OOP principles and layered architecture
