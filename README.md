# Student Management System

## Overview

A console-based Student Management System developed using Core Java, JDBC, and Oracle Database.

## Features

* Add Student
* View Students
* Search Student by ID
* Update Student Details
* Delete Student Records

## Technologies Used

* Core Java
* JDBC
* Oracle Database 10g
* SQL*Plus
* Eclipse IDE

## Concepts Implemented

* JDBC Connectivity
* PreparedStatement
* ResultSet
* CRUD Operations
* Exception Handling
* DAO Design Pattern
* Object-Oriented Programming

## Project Structure

com.jdbc.model
com.jdbc.dao
com.jdbc.util
com.jdbc.main
